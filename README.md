# travel-online

项目描述：  
基于用户旅游出行的实际需求，开发面向广大旅游爱好者的在线预订平台。项目包含 用户服务、景点管理、门票订单、酒店预订、评价系统、旅游攻略 等核心模块。功能涵盖景点浏览、门票购买、酒店预订、评价互动 等完整业务流程，旨在构建一个能够在大用户量的高并发环境下，具备高性能、高可用的在线旅游交易平台。  

技术栈：Spring Boot / MyBatis / MySQL / Redis / JWT / AOP / Vue  

•  该项目由B站「程序员辰星」制作  
•  目前本人在该项目源码基础上进行迭代修改，解决源码的部分问题；并加入Redis技术栈，解决超卖/并发安全问题  
•  项目仅作为个人项目的展示，不作为商业用途
